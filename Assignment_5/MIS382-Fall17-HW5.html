Question 1: Ensembles (1+12+2 = 15pts)¶

In this question, we will compare performance of different ensemble methods: Random Forest, Gradient Boosting Decision Tree (GBDT), and XGBoost. Note that you have to install xgboost package in addition to scikit-learn. You can see installation guides here.

Two datasets are provided for this problem. For each of the datasets ((X1.csv, y1.csv), (X2.csv, y2.csv)), do the following:

  1. Load the data and partition it into features (X) and the target label (y) for classification task. Then, use train_test_split to split data into training and testing: test_size=0.33, random_state=42.

  2. Build a classifier using Random Forest, Gradient Boosting Decision Tree, and XGBoost, respectively, and answer the following for each classifier.

    • Mention any design choices (with reasoning/justification) that you made, e.g. the hyperparameters considered for each classifier.
    • Report the mean error rate (fraction of incorrect labels) and the confusion matrix on test data.
    • Report the feature importance and time of execution (training and predicting times).

  3. Compare the three classifiers for the two different datasets ((X1.csv, y1.csv), (X2.csv, y2.csv)) in terms of the misclassification rate. What are the characteristics of the dataset and the classifiers that resulted in somewhat different comparative results?
